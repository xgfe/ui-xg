# button
## Description
按钮组件,可以展示不同样式和功能作用的按钮。

## Usage

``` html
<uix-button [loading="boolean"]
    [b-type="string"]
</uix-button>
```
## Restrict
- 'AE'

## Arguments

- loading(optional):指定是否提交加载,默认显示false
    - type:`boolean`
    - default:`false`
- bType(optional):指定按钮类型,可取值`button`、`reset`、`submit`,默认为`button`
    - type:`string`
    - default:`button`
